# Iris Flower Species Prediction using Machine Learning and Flask

## Project Overview
This is a beginner-friendly Machine Learning project that predicts the species of an Iris flower using four measurements:

1.Sepal Length
2.Sepal Width
3.Petal Length
4.Petal Width

The model is trained using the Iris dataset and deployed as a simple Flask web application.

## What Students Will Learn
--How to load a dataset
--How to split data into training and testing sets
--How to train ML models
--How to compare model accuracy
--How to save a trained model
--How to create a Flask web app for prediction

## Machine Learning Models Used
- Decision Tree
- Random Forest
- Logistic Regression
